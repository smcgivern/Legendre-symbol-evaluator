Properties of the Legendre symbol

p is an odd prime, a ≢ 0 (mod p) and b ≢ 0 (mod p), and q ≠ p is an odd prime.

(a)
If a ≡ b (mod p) then (a | p) = (b | p)
(b)
(a2 | p) = 1
(c)
(ab | p) = (a | p) (b | p)
(d)
(-1 | p) = (-1)(p - 1) / 2
(e)
(2 | p) = (-1)(p2 - 1) / 8
